# 公开源码范围说明

本仓库面向比赛提交，公开源码范围以主体任务代码、部署配置、演示脚本和设计说明为主。仓库采用 `PolyForm Noncommercial License 1.0.0`，允许学习、研究、测试、教学和非商用展示使用；商业使用需要额外书面授权。

## 已纳入

- ROS2 底盘、安全守护、传感器适配、建图相关代码。
- K1 端 D435 YOLO 风险识别和风险点地图化脚本。
- 本地 dashboard、演示启动脚本、收尾脚本。
- 本地 LLM 报告生成接口与配置。
- 机械臂 no-load 安全响应与动作语义配置。
- RL/RRT/MoveIt 相关动作空间、策略训练/评估脚本和接口说明。
- 量化 YOLOv8n ONNX 演示模型。
- 项目报告、样例地图、样例证据和硬件图片。

## 未纳入

- 原始训练数据集、采集视频、长时间运行输出。
- 大体积 ROS bag、数据库、临时日志。
- 历史开发 bundle、压缩包、缓存目录。
- 私有环境变量、密钥、token、账号信息。

## 原因

这些内容要么体积过大，要么包含环境相关或隐私相关信息，不适合作为 GitHub 主仓库的一部分。若需要复现实验，可根据 `docs/` 下的部署文档重新采集数据和生成输出。
