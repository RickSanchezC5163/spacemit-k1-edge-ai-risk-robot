#### 展示四份关键证据

**证据 1：底盘安全链路**
```
/input_cmd_vel → scan_safety_guard_node → /cmd_vel_guarded → C30D
hard_stop=0.30m, emergency_stop=0.20m
base_zero_ok = guarded_cmd_zero AND robot_vel_zero AND diag_zero
```

**证据 2：视觉规则检测（正例 + 负例）**
```
正例: red_object_detected=true, bbox=[93,250,275,117], depth=0.561m
负例: red_object_detected=false, red_mask_pixels=0
      → arm_execution_status=skipped_negative_control
      → 系统正确静默，不误触发
```

**证据 3：风险空间化**
```
camera bbox + depth → camera_link → base_link → odom → map
risk_point → Map-A0 投影 → risk_map_snapshot.png
```

**证据 4：机械臂安全边界**
```
7 层安全验证: 相位门 → 协议验证 → 关节限位 → 步长限制 → 工作空间 → 机器人安全 → 急停
global_gate AND phase_gate: 全局关闭时任何阶段无法绕过
serial_bytes_written=180 (单次 no-load), contact_allowed=false
```

## 四、视频录制建议

### 方案 A：单次 run 录制 + 后期画中画

1. OBS 开 4 个源:
   - 真机俯拍（手机/摄像头）
   - D435 实时画面（RViz2 或 image_view）
   - 终端日志（SSH 窗口，`CONSOLE_MODE=compact`）
   - RVIZ SLAM 地图（可选）

2. 跑一次 `--fast-demo` 模式

3. 后期加字幕标注关键事件

### 方案 B：分镜录制 + 剪辑

1. 单独录：底盘自主避障、D435 红色检测 overlay、机械臂动作序列、LLM 报告生成
2. 剪辑拼接，加解说旁白

**推荐方案 A**，因为一次 run 内所有事件自然串联，不需要额外解释"这是同一次"。
