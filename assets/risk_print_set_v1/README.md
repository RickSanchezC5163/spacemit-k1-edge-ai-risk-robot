# Risk Print Set V1

This directory is reserved for A4 printed risk-image simulation assets used in
the constrained-space demo scene.

Expected layout:

```text
assets/risk_print_set_v1/
  manifest.json
  samples/
```

No dataset files are committed here by default. Add only files that are
licensed for the project and record their source in `manifest.json`.

Claim boundary:

- Printed risk images simulate visual risk sources.
- They do not prove real-world defect detection accuracy.
- Server-side inference must not be described as K1 local inference.
